# 快速开始

> ciic-vue-ui 是基于[element-ui](https://element.eleme.io/) 二次封装的组件库！
> 里面有element-ui 原有的组件和新增ciic业务组件库

#### 安装组件

```bash

npm i ciic-vue-ui
cnpm i ciic-vue-ui

```
#### 引用组件库

```javascript

// 全局使用
import "ciic-vue-ui/dist/css/index.css";
import CiicVueUi from 'ciic-vue-ui';
Vue.use(CiicVueUi);


```
